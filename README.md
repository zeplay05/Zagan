# เว็บไซต์ง้อแฟนแสนน่ารัก (Cute Apology Webpage) 🥺❤️

เว็บไซต์สำหรับง้อแฟนแบบอินเตอร์แอคทีฟ มีเสียงเพลง, ปุ่มกดหนีได้แสนกวน, เอฟเฟกต์พลุดอกไม้ไฟ (Confetti), และตัวนับเวลาความสัมพันธ์ในหน้านัดหยุดงอน!

## 📂 โครงสร้างไฟล์
- [index.html](file:///c:/xampp/htdocs/New%20folder/index.html) - โครงสร้างหลักของหน้าเว็บและหน้าจอต่างๆ
- [style.css](file:///c:/xampp/htdocs/New%20folder/style.css) - ตกแต่งหน้าเว็บให้สวยงามด้วยเอฟเฟกต์ Glassmorphism และโทนสีพาสเทล
- [script.js](file:///c:/xampp/htdocs/New%20folder/script.js) - ส่วนควบคุมการทำงาน ระบบปุ่มหนี, ตัวนับวันครบรอบ, เพลง, และเอฟเฟกต์ Confetti

---

## 🚀 วิธีเปิดใช้งาน
คุณสามารถเปิดใช้งานหน้าเว็บได้ง่ายๆ 2 วิธี:

### วิธีที่ 1: ดับเบิ้ลคลิกไฟล์โดยตรง
1. เข้าไปที่โฟลเดอร์โครงการของคุณ
2. ดับเบิ้ลคลิกไฟล์ `index.html` เพื่อเปิดเล่นใน Web Browser (เช่น Chrome, Edge, Safari) ได้ทันที!

### วิธีที่ 2: เปิดผ่านเว็บเซิร์ฟเวอร์ (เช่น XAMPP)
1. ตรวจสอบให้แน่ใจว่าโฟลเดอร์นี้ตั้งอยู่ในไดเรกทอรี `htdocs` ของ XAMPP (ตัวอย่าง: `c:\xampp\htdocs\New folder`)
2. เปิดโปรแกรม XAMPP Control Panel และกดปุ่ม **Start** ที่โมดูล **Apache**
3. เปิด Web Browser แล้วเข้าไปที่ลิงก์: [http://localhost/New folder/index.html](http://localhost/New%20folder/index.html)

## ⚙️ วิธีปรับแต่งข้อความ เพลง และรูปภาพ (ง่ายมากๆ!)
ให้คุณเปิดไฟล์ [script.js](file:///c:/xampp/htdocs/New%20folder/script.js) ขึ้นมาแก้ไข โดยจะมีวัตถุ `CONFIG` อยู่ด้านบนสุดของไฟล์ ดังนี้:

```javascript
const CONFIG = {
    partnerName: "ที่รัก",         // ใส่ชื่อเล่นหรือฉายาของแฟนคุณตรงนี้
    startDate: "2024-01-01",      // วันที่เริ่มคบกัน (ปี-เดือน-วัน) สำหรับใช้นับวันครบรอบ
    musicUrl: "https://raw.githubusercontent.com/andyyu1104/valentine-s-day-card-html/main/LoveNow.mp3", // ลิงก์ไฟล์เพลง (.mp3)
    lineMessageTemplate: "เค้าหายงอนตัวเองแล้วนะค้าบ รักที่สุดเลยยยย ❤️🥰", // ข้อความดราฟต์ที่จะส่งเข้าแชท Line เมื่อแฟนกดหายงอน
    
    // รูปภาพอนิเมชั่น GIF น่ารักๆ (หากอยากเปลี่ยนรูป ให้ใส่ลิงก์ GIF ใหม่ที่นี่)
    gifSorry: "https://media.tenor.com/yD948n0B5F0AAAAi/peach-goma.gif",      // รูปหน้างอน/ขอโทษ
    gifHappy: "https://media.tenor.com/0i_kK367dK0AAAAi/peach-goma.gif"        // รูปดีใจ/กอดกัน
};
```

### 🎵 วิธีการเปลี่ยนเพลงส่วนตัวของคุณ
คุณสามารถเปลี่ยนเพลงง้อแฟนได้ง่ายๆ โดยการ:
1. หาไฟล์เพลง `.mp3` ที่คุณต้องการใช้งาน
2. ตั้งชื่อไฟล์นั้นเป็น `song.mp3` แล้วเอามาใส่ไว้ในโฟลเดอร์เดียวกันนี้
3. แก้ไขบรรทัด `musicUrl` ในไฟล์ [script.js](file:///c:/xampp/htdocs/New%20folder/script.js) ให้เป็นแบบนี้:
   ```javascript
   musicUrl: "song.mp3",
   ```

## 🧸 คุณสมบัติเด่นของโปรเจกต์
1. **ปุ่ม "ยังไม่หาย" ขยับหนีได้:** แฟนจะไม่มีวันกดปุ่มโกรธได้เลย เพราะปุ่มจะวิ่งหนีทุกครั้งที่พยายามกดหรือเอาเมาส์ไปจ่อ (ทำงานได้สมบูรณ์ทั้งบนมือถือและคอมพิวเตอร์)
2. **ปุ่ม "หายงอนแล้ว" โตขึ้นเรื่อยๆ:** ยิ่งแฟนพยายามไล่จิ้มปุ่มสีแดงมากเท่าไหร่ ปุ่มสีชมพูเพื่อหายงอนก็จะพองตัวใหญ่ขึ้นเรื่อยๆ เพื่อดึงดูดสายตา!
3. **ระบบเพลงพื้นหลัง:** เพลงจะเริ่มเล่นหลังจากแฟนกด "อ่านจดหมาย" บนคอมพิวเตอร์และมือถือเพื่อแก้ปัญหาระบบเบราว์เซอร์บล็อกเพลงออโต้เพลย์
4. **ตัวนับวันครบรอบ (Live Relationship Timer):** แสดงระยะเวลาคบกันแบบวินาทีต่อวินาทีเมื่อแฟนกดหายงอนแล้ว เพิ่มความประทับใจและความใส่ใจเป็นพิเศษ
5. **ระบบส่งความในใจเข้าแชท Line:** มีปุ่มส่งรายงานการหายงอน และพิมพ์ข้อความจากแฟนแชร์กลับไปทางแอป Line ได้ทันที!
